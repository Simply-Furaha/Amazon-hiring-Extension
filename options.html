Help & Tips

Getting Started:

  1. Navigate to the Amazon hiring page (hiring.amazon.ca/app#/jobSearch)
  2. Click the extension icon and press "Start Monitoring"
  3. The extension will automatically check for new jobs and apply based on your preferences

Tips for Best Results:

  • Keep the Amazon hiring tab open while monitoring
  • Set realistic preferences to avoid missing opportunities
  • Check the activity log regularly to see what's happening
  • Use a reasonable monitoring interval (10 seconds is recommended)

Troubleshooting:

  • If applications aren't working, try refreshing the Amazon page
  • Enable debug mode for more detailed logging
  • Make sure you're logged into your Amazon account
  • Check that browser notifications are enabled
